# Proyecto Final API

API REST desarrollada con Node.js y Express.

## Tecnologías

- Node.js
- Express
- Firebase Firestore
- Thunder Client

## Instalación

npm install

## Ejecución

npm start

## Endpoints

GET /api/products

GET /api/products/:id

POST /api/products/create

DELETE /api/products/:id
